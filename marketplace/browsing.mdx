## The Marketplace

The Ardie marketplace is your gateway to discovering knowledge bases. Browse by category, search by keyword, or filter by tags to find the right knowledge base for your needs.

## Browsing Options

### By Category

Navigate directly to a category to see all knowledge bases in that domain:

- [Education](https://ardie.ai/knowledge-bases/category/education)
- [Policy](https://ardie.ai/knowledge-bases/category/policy)
- [Health](https://ardie.ai/knowledge-bases/category/health)
- [Research](https://ardie.ai/knowledge-bases/category/research)
- [Legal](https://ardie.ai/knowledge-bases/category/legal)

### By Search

Use the global search bar to find knowledge bases by:
- **Title** — The name of the knowledge base
- **Publisher** — The organization that created it
- **Keywords** — Terms in the description or coverage notes

### By Tags

Filter by specific tags to narrow results. Tags indicate specific topics, industries, or use cases covered by a knowledge base.

## Sorting Options

Sort your results by:

| Sort Option | Description |
|-------------|-------------|
| **Popular** | Most subscribers and query volume |
| **New** | Recently added to the marketplace |
| **Updated** | Most recently updated content |

<Tip>
  Start with "Popular" to see well-established, actively-used knowledge bases.
</Tip>

## Knowledge Base Details

Click on any knowledge base to view its detail page, which includes:

- **Full description** and coverage notes
- **Publisher information**
- **Document and vector counts**
- **Available pricing plans**
- **Developer panel** with API examples

## Evaluating a Knowledge Base

Before purchasing, consider:

<Steps>
  <Step title="Check Coverage">
    Read the description and coverage notes to ensure the knowledge base covers your topic area.
  </Step>
  <Step title="Review Metrics">
    Look at subscriber count and query volume as indicators of quality and active maintenance.
  </Step>
  <Step title="Check Update Frequency">
    The "Updated At" date shows how recently the content was refreshed.
  </Step>
  <Step title="Review Plans">
    Compare available plans to find one that matches your expected query volume.
  </Step>
</Steps>
