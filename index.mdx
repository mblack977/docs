## What is Ardie?

Ardie is a **knowledge infrastructure platform** that connects you to curated, queryable knowledge bases via API. Browse the marketplace, purchase access to specialized knowledge bases, and integrate them into your applications using scoped API keys.

## Key Features

<CardGroup cols={2}>
  <Card title="Scoped API Keys" icon="shield-check">
    Every key is Ardie-issued, scoped to a specific knowledge base, and revocable. You never receive supplier infrastructure keys.
  </Card>
  <Card title="Prepaid Usage Model" icon="gauge">
    Plans include a fixed number of queries per billing period. No surprise bills—you always know your limits.
  </Card>
  <Card title="Citations Included" icon="quote-left">
    Every response includes citations with document titles, excerpts, and page numbers for verification.
  </Card>
  <Card title="API-First Design" icon="code">
    Pure REST API access—integrate knowledge bases into any application, workflow, or automation.
  </Card>
</CardGroup>
